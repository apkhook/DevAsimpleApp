# 什么鬼

结合OkHttp，RxVolley，下拉刷新，短信验证自动识别等等一步一步的开发一个简单的多功能app

借鉴了一些开源的代码，进行了修改重组以及自己的还有不少自己未完成的功能

感谢所有的开源作者
